# FamilyCare Senior V1.0.66 Render update

1. Open GitHub repository `familycare-senior`.
2. Upload all files from this archive directly in the repository root, over the existing files.
3. Commit changes.
4. In Render, open `familycare-senior`.
5. Run `Manual Deploy -> Clear build cache & deploy`.
6. On the phone, close/reopen the PWA. If the old layout remains, uninstall and reinstall the PWA from Chrome.

No database script is required. No Environment Variables need to be changed.
